Laboratory Work #4

Software Application Development

Assignment:

  1. Create a Russian version of the site ru/index.html.
  2. Create an English version of the site en/index.html.
  3. Create index.html in the project root for general information, which should have navigation to the previously added pages.
  4. Configure GitHub Pages for automatic build from the gh-pages branch.
  5. Implement CI/CD that publishes the application on GitHub Actions when a new release is created. The application should be available at {your-githubpages-address.io}/your-repository/v{your-release-name}/
  6. Ensure availability of previous versions of the site.
  7. Prepare a report that must include:
    1. Attach screenshots of GitHub Pages settings with a description of what each setting is for and how it will work.
    2. Screenshot of CI/CD workflow with a description of each step.
    3. Screenshots of the web application showing the browser address bar.
    4. Pipeline code.
